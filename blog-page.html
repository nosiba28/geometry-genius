What is DOM and what is the purpose of DOM?

DOM (Document Object Model) is a application programming interface which is used for web documents.
The purpose of DOM is given below:

  • It represents the document as nodes and objects
  • It helps so that programming languages can interact with the page
  • It is used to change the document structure
  • It is used to change the document style
  • It is used to change the document content

How will you select HTML elements using DOM? Name the DOM methods.

We will select HTML elements using DOM selectors. There are some DOM methods to select HTML elements:

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

What is event bubble?

Event bubble is a related term to DOM. It happens when an element receives an event, and that event is propagated to its parent and ancestor elements in the DOM tree until it gets to the root element.

What is a callback function and why will you use it?

A callback function is a function that is passed into another function as an argument, which is then called inside the other function to complete other execution. Call back function only run when the current running function's task has completed. It is used for waiting for previous function call and then execute another function call.
